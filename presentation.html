=>
The fat arrow => replaces the keyword function.
Arrow functions can have implied return statements.
Arrow functions allow optional parenthesis when there's only one parameter.
value => {...}
Parenthesis are still required if there's no parameters or multiple parameters.
(value, i) => {...}
() => {...}
